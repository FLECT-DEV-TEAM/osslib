&{"overview"}

#{if lang == "ja"}

Slugコンパイル時に「grunt heroku」を実行するBuildpackです。

単体では動作しないのでheroku-buildpack-multiを使用して他のbuildpackと組み合わせて使用します。

#{/if} #{else}

Run "grunt heroku" command at slug compile.

It doesn't do anything else, so to actually compile your app you should use heroku-buildpack-multi to combine it with a real buildpack.

#{/else}

&{"usage"}

#{if lang == "ja"}

このbuildpackのURLと自分の使いたいオリジナルのbuildpackのURLを「.buildpacks」ファイルに記述してプロジェクトのルートディレクトリにおきます。

Herokuアプリケーション作成時にはbuildpackにheroku-buildpack-multiを指定してください。

#{/if} #{else}

To use this buildpack, you should prepare .buildpacks file that contains this buildpack url and your real buildpack url.

#{/else} 
    $ ls
    .buildpacks
    ...
    
    $ cat .buildpacks
    https://github.com/shunjikonishi/heroku-buildpack-with-grunt-task
    https://github.com/heroku/heroku-buildpack-play
    
    $ heroku create --buildpack https://github.com/ddollar/heroku-buildpack-multi
    
    $ git push heroku master
    ...
#{if lang == "ja"}

Gruntの実行結果を確認するには以下のコマンドでDynoのシェルに入ってください。

#{/if} #{else}

You can verify your grunt task result by following command.

#{/else} 
    $ heroku run bash
    ...
