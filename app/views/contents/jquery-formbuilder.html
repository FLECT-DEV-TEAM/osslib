&{"usage"}

#{if lang == "ja"}

内容を生成するform要素に対してフォーム定義のJSONを引数にしてメソッドを実行するだけです。

#{/if} #{else}

You can get web form by only executing formbuilder method with JSON definition.

#{/else} 
    $("#form").formbuilder({
        "items" : {
            "name" : {
                "type" : "text",
                "required"
            },
            ....
        }
    });
