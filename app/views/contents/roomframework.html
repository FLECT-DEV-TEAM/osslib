roomframeworkはWebSocketアプリケーションを作成するためのフレームワークです。

クライアントサイドのJavaScriptライブラリであるroomframeworkとPlay2用のScalaライブラリであるroomframework-playが連携して、Chatのような一つのルームを複数人が共有するタイプのアプリケーションを簡単に作成できます。

Salesforce1 Developers Community MAXというイベントでWebSocketについて講演した際の資料をこちらで公開しています。

#{/if} #{else}

Roomframework is a framework for WebSocket applications.

It consists of two parts.

  • roomframework - JavaScript library for client.
  • roomframework-play - Scala library for playframework.
