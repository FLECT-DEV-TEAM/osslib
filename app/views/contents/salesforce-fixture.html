#{if lang == "ja"}

YAMLで定義したレコード情報に従ってSalesforce上のデータを作成／更新／削除するアプリケーションです。

Salesforceアプリケーション開発時にテストデータを繰り返し同じ状態に戻したい場合などに活用できます。

#{/if} #{else}

This application allows you to create/update/delete data on Salesforce by YAML definition.

You can use this app to maintain testdata.

#{/else}

&{"install"}

    git clone git@github.com:shunjikonishi/salesforce-fixture.git
    cd salesforce-fixture
    heroku create
    git push heroku master
    heroku config set SALESFORCE_APPID=<SALESFORCE OAUTH_CONSUMER_KEY>
    heroku config set SALESFORCE_SECRET=<SALESFORCE OAUTH_CONSUMER_SECRET>
